Estas instâncias artificiais não têm um problema concreto. Cada estado é abstrato e é representado por um inteiro. O primeiro estado é o estado 1. No entanto, cada instância tem diferentes características\+:
\begin{DoxyItemize}
\item Ramificação -\/ existe valor mínimo e máximo, pelo que pode-\/se controlar se é elevada ou baixa, fixa ou variável.
\item Nível do objetivo -\/ pode-\/se definir o nível mínimo em que o objetivo pode estar, e assim indicar que é 8. Nesse caso apenas estados no nível 8 ou superior, é que podem ser objetivo.
\item Nível máximo -\/ a profundidade máxima em que há estados. Após este nível os sucessores ficam vazios.
\item Objetivo -\/ permite controlar a raridade do objetivo. Define-\/se um número K, se ID é objetivo então tem de ser multiplo de K. No caso de K=10, significa que todos os estados que tiverem ID a terminar com 0, são objetivo.
\item Custo máximo -\/ permite controlar entre custos das ações fixos, ou variáveis
\item Repetição de estados -\/ define-\/se um número K, ficando todos os IDs resto da divisão por K. Assim, controla-\/se a probabilidade de um estado ser repetido, K menor há mais repetidos.
\item Repetição de estados mas por nível -\/ idêntico ao anterior, mas os estados apenas são repetidos de entre os que estão no mesmo nível.
\end{DoxyItemize}

Estas caracteristicas definem o espaço de estados, e permite avaliar os algoritmos de forma independente dos problemas, ou testar em situações extermas, mesmo antes de ter um problema concreto que tenha essas características.
