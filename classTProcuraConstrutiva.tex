\doxysubsection{Detailed Description}
Representa um estado no espaço de estados. 

Esta classe base deve ser redefinida com um problema concreto, permitindo a execução de procuras construtivas.

Para utilizar a classe, é necessário\+:
\begin{DoxyItemize}
\item Redefinir os métodos de duplicação e cópia de estados (\doxylink{group__RedefinicaoMandatoria_gaf7c2483f38d11825c220b7a8d538a623}{Duplicar()} e \doxylink{group__RedefinicaoMandatoria_ga92eb7af76e68a20fe542afb459e7d642}{Copiar()}).
\item Implementar funções essenciais de procura\+: \doxylink{group__RedefinicaoMandatoria_ga687ad49ea5304ae8e2c3782f470c4ad3}{Solucao\+Vazia()}, \doxylink{group__RedefinicaoMandatoria_ga1f3e2b667881c3e910ba372467645304}{Solucao\+Completa()}, \doxylink{group__RedefinicaoMandatoria_ga0a13570d2f5a21296e3c10c2780e170c}{Sucessores()} e \doxylink{group__RedefinicaoSugerida_ga6bb2ece76e931695c9e32df7ca7143aa}{Debug()}.
\end{DoxyItemize}

Permite execução dos algoritmos\+:
\begin{DoxyItemize}
\item {\bfseries{Cegos}}\+: \doxylink{group__ProcurasCegas_gafe656922894d8a27b4cfb70128793af0}{Largura\+Primeiro()}, \doxylink{group__ProcurasCegas_ga2343fafdecfd55b69c0ffb71e8f9426b}{Custo\+Uniforme()}, \doxylink{group__ProcurasCegas_gaf00d9776f6b25d56138efb6725b86228}{Profundidade\+Primeiro()}
\item {\bfseries{Informados}}\+: \doxylink{group__ProcurasInformadas_gaebb84182343041bc276e4ceceb4b13aa}{Melhor\+Primeiro()}, \doxylink{group__ProcurasInformadas_ga37dd78e1a9a6742700728135f81964b4}{AStar()}, \doxylink{group__ProcurasInformadas_gae77f9608e7988052377d7025bee7c4e8}{IDAStar()}, \doxylink{group__ProcurasInformadas_gafe804dbf6cabda9e627500c2d4f8aafd}{Branch\+And\+Bound()}
